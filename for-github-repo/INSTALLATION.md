# Otter Order Consolidator - Tablet Installation

## Quick Install (Recommended)

### Step 1: Install Tampermonkey
- **iPad/iPhone**: [App Store Link](https://apps.apple.com/app/tampermonkey/id1482490089)
- **Android**: Use Firefox or Kiwi Browser, then install Tampermonkey extension

### Step 2: Install the Script

#### Option A: Use Mini Installer (Easiest)
1. Click this link on your tablet: [mini-installer.user.js](https://raw.githubusercontent.com/liwesley02/otter-add-on/main/mini-installer.user.js)
2. Tampermonkey should prompt to install
3. Click "Install" 
4. Go to https://app.tryotter.com - the full script will auto-download

#### Option B: Direct Install
1. Click this link: [otter-order-consolidator.user.js](https://raw.githubusercontent.com/liwesley02/otter-add-on/main/otter-order-consolidator.user.js)
2. If Tampermonkey doesn't auto-prompt:
   - Copy the URL
   - Open Tampermonkey dashboard
   - Click "Utilities" tab
   - Paste URL in "Install from URL"
   - Click "Install"

## Usage
1. Go to https://app.tryotter.com
2. Login to your account
3. Navigate to Orders page
4. The consolidator overlay will appear automatically

## Troubleshooting
- **Not working?** Make sure Tampermonkey is enabled for tryotter.com
- **No overlay?** Refresh the page
- **Installation failed?** Try the mini-installer method instead
